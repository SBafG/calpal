# Screenshots

Reference captures of the Cal·Pal v3 design at 1480 × 535 px (preview-sized).
Open the live prototype at `../prototypes/Cal·Pal v3.html` for the full page
and interactive Tweaks panel.

| File | Shows |
|---|---|
| `01-hero-spring.png`   | Top of page — TopStrip, Masthead, TodayStats (default state: Vår / Spring, brick accent) |
| `02-calendar.png`      | The 6-week calendar grid + start of sidebar (sun arc, pollen) |
| `03-quote-sidebar.png` | Month essay quote + sidebar bottom (säsong / dagens ord) |
| `04-footer.png`        | Footer |
| `05-season-summer.png` | Hero in Sommar palette (ochre accent, `#d68a18`) |
| `06-season-autumn.png` | Hero in Höst palette (rust accent, `#8a3a1a`) |
| `07-season-winter.png` | Hero in Vinter palette (ink-blue accent, `#2a4a6a`) |

The seasonal screenshots (05–07) demonstrate how `data-season` on the page
root swaps the background tint and accent color without touching any other
structural color. The brick `#c44a2a` in 01 is the Vår default.
